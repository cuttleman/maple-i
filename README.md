# **Maple I**

## Get started

First install maplei:

```
npm install -g maplei
```

## Usage

```
maplei <option>
```

|  option   |  description  | default | require |
| :-------: | :-----------: | :-----: | :-----: |
| -n --nick | user nickname |    -    |   🟢    |
| -p --path | download path |   cwd   |   🔴    |

<br />

```js
// example
maplei -n 타락파워전사 -p Users/<username>/Desktop
```
